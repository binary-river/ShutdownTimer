## Install
Main.java 와 ShutdownTimer.java 파일로 이루어진 프로그램입니다.<br>
해당 파일을 컴파일 및 실행하시거나, manifest.txt 를 이용해 jar 파일을 생성하여 사용 가능합니다.

## Usage
START를 클릭하는 경우, 입력한 시간이 지나면 컴퓨터가 자동으로 종료됩니다.<br>
CANCEL 버튼을 클릭하면 종료 예약이 취소됩니다.<br>
